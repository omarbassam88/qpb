# Quick Project Builder

A terminal Command Line interface to quickly scaffold different projects with standard tools and architecture.

## Installation

``` sh
npm i -g qpb
```

## Usage

``` sh
qpb
```

## Templates

[] Node JS
    [] CLI
    [] Server
[] Frontend
    [] vanilla JS/TS
    [] React
    [] Vue
    [] Svelte
    [] preact
    [] Alpine
    [] SolidJS
[] Desktop 

## Development Setup

``` sh
pnpm install
```
